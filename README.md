Project Overview

This project is a requirement for the Web Design and Development course, demonstrating mastery of the following modern front-end concepts:

Component Architecture: Built using React.js with reusable components (Navbar, Header, SocialLinks).

Data Management: Uses JavaScript Array Map function to dynamically render navigation and social links from dedicated data files.

Responsive Design: Implements CSS Media Queries to ensure a clean layout on both desktop and mobile screens.

Design System: Spacing and sizing adhere to the 8-Point Grid System.

Technology Stack Used

Framework: React.js

Styling: CSS3

Iconography: FontAwesome CDN
